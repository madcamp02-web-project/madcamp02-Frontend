## 🔧 핵심 정합 사항

### API 엔드포인트 (9개 컨트롤러)
- `/api/v1/auth/*` - 인증
- `/api/v1/user/*` - 사용자
- `/api/v1/trade/*` - 거래
- `/api/v1/stock/*` - 주식
- `/api/v1/watchlist/*` - 관심종목
- `/api/v1/game/*` - 게임/가챠
- `/api/v1/chat/*` - AI 상담
- `/api/v1/calc/*` - 배당금 계산
- `/api/v1/notifications/*` - 알림

### 데이터베이스 (9개 테이블)
`users` → `wallet` → `portfolio` → `trade_logs` → `items` → `inventory` → `chat_history` → `watchlist` → `notifications`

### WebSocket 채널 (4개)
- `/topic/stock.{ticker}` - 시세 브로드캐스트
- `/user/queue/trade` - 체결 알림
- `/user/queue/notification` - 개인 알림
- `/topic/ranking` - 랭킹 변동
